Why use Site Starter?

  1. Embed best practices

    Site Starter embeds the HTML5 Boilerplate project — so you're building off of “rock-solid defaults”.

  2. Don't repeat yourself

    Site Starter makes it easy to create layouts and site-wide headers, footers and other includes without having to copy them across every page.

  3. Established Growth Path

    Being built off Django Templates you can graduate straight to a full Django project when you needs expand.

So what is it?

Site Starter is a set of simple templates and tools to let you build a website (on a sound basis) quickly.
